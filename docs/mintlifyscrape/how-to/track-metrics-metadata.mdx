Logging metrics and metadata is standardized in ZenML. The most common pattern is to use the `log_xxx` methods, e.g.:

* Log metadata to a [model](/how-to/track-metrics-metadata/attach-metadata-to-a-model): `log_model_metadata`
* Log metadata to an [artifact](/how-to/track-metrics-metadata/attach-metadata-to-an-artifact): `log_artifact_metadata`
* Log metadata to a [step](/how-to/track-metrics-metadata/attach-metadata-to-steps): `log_step_metadata`
