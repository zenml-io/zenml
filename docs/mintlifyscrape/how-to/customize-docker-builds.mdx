ZenML executes pipeline steps sequentially in the active Python environment when running locally. However, with remote [orchestrators](/user-guide/production-guide/cloud-orchestration) or [step operators](/stack-components/step-operators), ZenML builds [Docker](https://www.docker.com/) images to run your pipeline in an isolated, well-defined environment.

This section discusses how to control this dockerization process.
